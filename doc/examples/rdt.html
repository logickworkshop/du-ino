DU-RDT

The DU-RDT (a clone of the hardware DU MDLR module of the same name) is a Swiss army knife of sequential logic functions on a common clock input: a 16-bit sampling shift register which operates as a 16-bit LFSR when the input is disconnected, with two selectable outputs from the top and bottom bytes as well as a pattern lock switch; a D-type flip flop with normal and complemented outputs; and a T-type flip flop normaled to the D-type's complemented output.

Upper LFSR Out

Output from the selected bit of the upper byte of the shift register.

Lower LFSR Out

Output from the selected bit of the lower byte of the shift register.

Clock In

Primary clock input.

1s In

Feeds 1s into the shift register when active.

Pattern Loop

Feed a gate or turn the knob clockwise to lock and loop the current shift register pattern.

D In

Gate input for the D flip flop (set knob to -10V).

T In

Gate input for the T flip flop (set knob to -10V).

Clock Out

Primary clock output.

D Out

D flip flop output (divides clock by 2 if D input is zero).

Inverted D Out

Inverted D flip flop output.

T Out

T flip flop output (divides clock by 4 if D and T inputs are zero).

Save Box

Double-click to jump to the save box. Click to save the current envelope values to memory.

Clock Rate

Scroll to modify the internal clock rate, in beats per minute (up to 300 BPM). Displays EXT if overridden by the external clock input.

Swing

Scroll to modify the amount to swing the clock beat (starting from 50% straight time), up to 74% in increments of 6%.

Loop Indicator

Inidcates that the current pattern is locked and looping when visible.

LFSR Bit Pattern

Displays the 16-bit pattern currently stored in the linear feedback shift register.

Upper LFSR Bit

Scroll to select the bit to output from the upper (left) byte of the shift register. The left arrow under the pattern display corresponds to this setting, indicating the current bit.

Lower LFSR Bit

Scroll to select the bit to output from the lower (right) byte of the shift register. The right arrow under the pattern display corresponds to this setting, indicating the current bit.

Upper LFSR Out Indicator

Displays the state of the upper LFSR bit output.

Lower LFSR Out Indicator

Displays the state of the lower LFSR bit output.

1s In Indicator

Displays the state of the 1s input.

D In Indicator

Displays the state of the D input.

T In Indicator

Displays the state of the T input.

D Out Indicator

Displays the state of the D output.

Inverted D Out Indicator

Displays the state of the inverted D output.

T Out Indicator

Displays the state of the T output.
