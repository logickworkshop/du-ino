DU QNTZR

The QNTZR function is a digital CV pitch quantizer, configurable to any semitone-based scale (with 31 handy scale presets), and with selectable key. It can be set to continuously quantize or to quantize on a trigger (i.e. quantized sample-and-hold), and has a trigger output for note change.

Note Change Trigger Out

Outputs a trigger whenever the current output note changes.

Trigger In

If the trigger setting is enabled, triggers a sample for quantization.

Pitch CV In

Unquantized pitch CV input.

Pitch CV Out

Quantized pitch CV output.

Trigger Mode

Scroll clockwise to enable trigger mode (lifted T icon), counter-clockwise to quantize continuously.

Key

Scroll to set the key (root note) of the scale.

Scale Preset

Scroll to select the scale preset. Will automatically update to display the corresponding scale preset when semitones are manually enabled/disabled (blank if no corresponding preset).

Scale Display

Scroll to navigate, click to enable/disable a semitone. Enabled semitones are indicated by a box on the corresponding key. The current output note is indicated by the highlighted key.
