DU VSEG

The VSEG (vector synth envelope generator) is a 4-point envelope generator based on the one found in the Sequential Circuits Prophet VS. Each point has an amplitude from 0V to 10V, and a duration from 11ms to 40s of the linear progression to the next point. A defined segment of the envelope can be repeated or ping-ponged a set number of times before sustaining, or continuously until the gate is released.

Gate In

Opens the envelope gate.

Retrigger In

Retriggers the envelope.

VCA Audio In

Audio input for the voltage-controlled attenuator.

10V Envelope Out

Envelope output with a peak at 10V.

5V Envelope Out

Envelope output with a peak at 5V.

VCA Audio Out

Audio output for the voltage-controlled attenuator.

Save Box

Double-click to jump to the save box. Click to save the current envelope values to memory.

Current Point

Indicates the envelope point currently being edited.

Point Level

Scroll to set the amplitude of the current envelope point. Click to move to the rate.

Point Rate

Scroll to set the rate (time to next point, in milliseconds) of the current envelope point. Click to move to the level of the next point.

Repeat/Ping-Pong

Scroll to select among different repeat or ping-pong patterns.

Repeat Count

Scroll to select the number of times to repeat or ping-pong (after which point 3 becomes a sustain). Select "C" for continuous.

Gate Indicator

Active when the gate is open.

Envelope Display

Displays the general shape of the envelope.
