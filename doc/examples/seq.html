DU-SEQ

The DU-SEQ (a clone of the DU MDLR hardware module of the same name) is an 8-stage pulsing sequencer with gate pattern, stage steps, and slew limiter assignable to each stage. With six gating patterns, including two external gate inputs, a huge diversity of sequencing patterns is possible.

Gate Out

Sequencer gate output.

Clock Out

Primary clock output.

Clock In

Primary clock input. Internal clock is used when disconnected.

Reset In

Reset trigger input. Immediately resets the pattern (and internal clock, if using).

Reverse/Address In

In direct mode, reverses the sequence direction when gate active. In address mode, CV selects the next stage (0V to 5V range).

Half Clock In

Divides the clock by 2 when gate active.

External Gate 1 In

Gate input for the external gate 1 gate pattern.

External Gate 2 In

Gate input for the external gate 2 gate pattern.

Pitch CV Out

Sequencer pitch CV output (1V per octave).

Save Box

Double-click to jump to the save box. Click to save the current envelope values to memory.

Stage Count

Scroll to select the number of stages to count (1 to 8) when sequencing. Click to jump to direct/address mode.

Direct/Address Mode

Set to D for direct mode (CI1 gate controls direction), A for address mode (CI1 CV selects next stage). Click to jump to slew rate.

Direction/Address Indicator

Indicates the current direction or stage selection (from CI1).

Half Clock Indicator

Indicates that the clock is divided by 2 (from CI2) when visible.

Slew Rate

Scroll to set the slew rate for stages with the slew rate limiter enabled. Click to jump to gate time.

Gate Time

Scroll to set the gate time (portion of a step during which the gate is high). Click to jump to clock rate.

Clock Rate

Scroll to modify the internal clock rate, in beats per minute (up to 300 BPM). Displays EXT if overridden by the external clock input. Click to jump to stage count.

Stage/Gate Cursor

Indicates the current output stage and gate output.

Stage Pitch

Scroll to select the pitch note for the stage. Click to cycle through stages. Double-click to jump to step count.

Step Count & Gate Pattern

Scroll to select the number of steps for the stage to count (left) or the gate pattern for the stage (right). Click to cycle through stages. Double-click to jump from step count to gate pattern, or from gate pattern to slew rate limiter.

Slew Rate Limiter

Scroll clockwise to enable the slew rate limiter for the stage, counter-clockwise to disable. Click to cycle through stages. Double-click to jump to top menu.
